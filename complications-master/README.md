# complications

A customizable watchface with 4 slots for different functions, with complete color customization.

The list of supported complications were:
* seconds
* month
* day dial
* weekday
* day box
* battery gauge
* bluetooth monitor
* battery icon
* day name
* digital time
* digital date
* pebble logo
